# shadcn/ui

Component library: TypeScript, Tailwind CSS, Radix UI. Open Code, AI-Ready.

## Adding Components

```bash
npx shadcn@latest add <component-name>
```

## Key Components

**Form & Input:** Form, Button, Input, Select, Checkbox, Switch, Calendar, Date Picker, Combobox

**Layout:** Accordion, Tabs, Sidebar, Scroll Area, Resizable

**Overlays:** Dialog, Sheet, Drawer, Popover, Tooltip, Dropdown Menu, Command

**Feedback:** Alert, Toast, Progress, Skeleton, Badge

**Display:** Avatar, Card, Table, Data Table, Chart, Carousel
